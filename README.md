## 2. Completed Steps
- Successfully downloaded and installed Cursor IDE and Git for Windows.

## 3. Issues Encountered and Solutions
- Issue 1: The system initially failed to recognize the git init command, throwing a red error (The term 'git' is not recognized).
  - Solution: Identified that Git was missing on the local machine. Downloaded the Git setup for Windows from the official website and restarted Cursor IDE to update system environment variables.
- Issue 2: Attempted to execute Linux-specific commands (rm -rf) inside the Windows PowerShell environment.
