# Nhật Ký Thực Hiện Dự Án

## 1. Các công cụ đã cài đặt
- *Cursor IDE*: Trình soạn thảo mã nguồn tích hợp AI.
- *Claude Code Extension*: Tiện ích mở rộng hỗ trợ lập trình thông minh của Anthropic.
- *Codex Extension*: Tiện ích hỗ trợ được tích hợp của OpenAI.
- *Git*: Công cụ quản lý phiên bản mã nguồn trên Windows.

## 2. Các bước đã hoàn thành
- Tải và cài đặt thành công Cursor IDE và Git cho Windows.
- Thiết lập mô hình AI Claude và OpenAI Codex trong môi trường làm việc.
- Tạo kho lưu trữ công khai (Public Repository) trên GitHub.
- Đồng bộ hóa thư mục cục bộ my-cursor-project với GitHub bằng Terminal.
- Tạo và viết tài liệu báo cáo vào tệp tin README.md.

## 3. Lỗi gặp phải và cách giải quyết
- Lỗi 1: Máy tính ban đầu không nhận diện câu lệnh git init (báo chữ đỏ).
  - Cách xử lý: Do máy chưa cài phần mềm Git. Đã truy cập trang chủ Git để tải bản cài đặt cho Windows, sau đó khởi động lại phần mềm Cursor để cập nhật môi trường hệ thống.
- Lỗi 2: Gõ nhầm lệnh hệ điều hành Linux (rm -rf) trên môi trường PowerShell của Windows.
  - Cách xử lý: Đổi sang lệnh Remove-Item chuyên dụng của Windows để làm sạch dữ liệu cũ.
